# Arquitectura del Proyecto

El proyecto está basado en una arquitectura modular utilizando scripts en C# dentro de Unity.

## Sistemas principales

- Sistema de jugador
- Sistema de enemigos
- Sistema de score
- Sistema de UI
- Sistema de mecánicas especiales

## Flujo general

Jugador:Interactúa con entorno, Recibe daño o recolecta, Se actualiza score, Condición de victoria/derrota
